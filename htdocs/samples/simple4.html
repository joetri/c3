AGT Data 2018-05-03

Readings taken every minute. X axis is number of minutes, Y axis is temp in degrees F. O2 levels are multiplied by 10 for smooth display in this Y range. All values are rounded cause who likes decimals ;-)

This is interactive and allows you to drill down on what you want. Hover over one of the X axis labels. Click on one of the X axis labels. Hover over any data point you're interested in in the chart.

Note that this is for demo purposes at this point. For more on what's possible with this approach see C3.js. I could get time labels on the x axis and lots of other tuning could happen. At this point I think it's roughed in enough to guage base fuctionality as an option moving forward.

Also note the responsive nature of this chart. View it in a smaller screen size and it auto shrinks. Also the system auto redraws the axis tick values. If we had a value that reached up to 250, the chart would accomodate that on the fly. There's a lot of built in smarts in this charting system: that's good if it fits your needs, bad if it doesn't. Note that the axes auto scaled to accomodate the values I put in and so you can throw any range of values (within reason) into the system and it will give you a smooth chart build automatically.
